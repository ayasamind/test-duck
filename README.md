# README

* Ruby 2.4.3
* Rails 5.1.5

* System dependencies
  - npm 3.10.10
  - yarn 1.3.2
  
## Development

### Setup

```
# ruby gems
$ bundle install --path vendor/bundle

# npm packages
$ yarn install

# db
# bundle exec rails db:create db:migrate
```


### Dev Server

```
$ bundle exec foreman start
```
